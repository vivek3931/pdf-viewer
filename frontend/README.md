# 🖥️ Nexus PDF Frontend

This is the high-performance React frontend for Nexus PDF. It leverages Vite for ultra-fast development and React 19 for the latest features.

## 🚀 Key Technologies
- **React 19**: Modern UI library.
- **Vite 6**: Fast build tool.
- **Tailwind CSS**: Utility-first styling.
- **Shadcn UI**: Premium component library.
- **react-pdf**: Robust PDF rendering.
- **Zustand**: Lightweight state management.

## 📦 Installation
```bash
npm install
```

## 🛠️ Development
```bash
npm run dev
```

## 🏗️ Production Build
```bash
npm run build
npm run preview
```

## 🎨 UI Features
- **Zero-Flicker Zoom**: Double-buffered rendering.
- **Responsive Sidebar**: Navigation and bookmarks.
- **Floating Controls**: Minimalist toolbar that hides on scroll.
- **Pinch-to-Zoom**: Native mobile-like experience.
- **Dark Mode Support**: Sleek, high-contrast interface.
